## Requirements

- Node 11.12.0
- NPM 6.7.0

## Development

#### Installing Dependencies

From within the root directory:

```sh
$ npm install
```

#### Build and Serve App
From root directory
```
$ npm run-script dev
```

## View Application
For development, in your browser, navigate to
```sh
http://localhost:8080
```

View the latest deploy at: https://the-matching-game.herokuapp.com/
