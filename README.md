> Have fun testing your memory.

## Requirements

- Node 18.13.0
- Yarn 1.22.19

## Development

#### Installing Dependencies

From within the root directory:

```sh
$ yarn install
```

#### Build and Serve App
From root directory
```
$ yarn dev
```

## View Application
For development, in your browser, navigate to
```sh
http://localhost:3000
```

View the latest deploy at: https://the-matching-game.herokuapp.com/
